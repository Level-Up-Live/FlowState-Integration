# Level Up Live x FlowState Integration Walkthrough

Frontend-only product walkthrough showing FlowState POS lane assignment handing off to a Level Up Live shooter-facing lane screen for an Action Target / Genesis-style range lane system.

The interface is optimized for a horizontal tablet at the lane or staff station.

## Run locally

```bash
pnpm install
pnpm dev
```

The app uses local React state only and is designed for sales/product walkthroughs:

- FlowState POS lane assignment flow
- Action Target / Genesis lane screen handoff
- Camera-assisted scoring explanation
- Bluetooth audio prompt connection flow
- Level Up Live player account sync
