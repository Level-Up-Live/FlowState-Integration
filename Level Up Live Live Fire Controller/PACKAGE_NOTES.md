# Level Up Live Live Fire Controller

Packaged frontend demo for the FlowState POS to Level Up Live lane-screen integration.

## Contents

- `src/` React source
- `public/` logo, profile, and range assets
- `dist/` current production build
- `package.json` and lockfile for local install/build

## Run Locally

```bash
pnpm install
pnpm dev
```

## Preview Built Version

```bash
pnpm preview
```

This is a frontend-only demo. FlowState POS, Action Target lane controls, camera scoring, Bluetooth audio, and account sync are simulated.
